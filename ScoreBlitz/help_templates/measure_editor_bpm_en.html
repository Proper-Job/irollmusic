Beats per Minute

This setting normally does not need to be changed.

You use this setting to enter tempo variations within the piece.
For instance, if you have a Piu Mosso section, i.e. a section that is faster than the rest of the piece, you would give the measures in that section a slightly higher bpm value.
However, this is usually not the case, so normally you do not need to change the Beats per Minute default value.

Use the Metronome in the top toolbar of the Roll by Measure performance style to effectively control the playback speed.
