Measure Editor Help
In order to play back your score using the Roll by Measure performance style you need to use the Measure Editor to enter Measures.

Go through the score and tap your finger onto each measure once. This will associate a marker with the Measure on the page. Tap the marker to change measure properties such as Tempo, Repeats, Endings and much more.

To move a marker press and hold it then drag it to the desired location.

Tap on any toolbar button to get more information on the functions that it performs.
