Annotations Editor Help
The Annotations Editor enables you to add Text, Freehand as well as Sign annotations to your score.

Select the Pen Tool to draw on the screen using your finger. The Pen button let's you change line width and color.

Select the Text tool to place text annotations anywhere on the page.

Select the Erase tool to use your finger to erase unwanted freehand annotations.

Select the Sign tool to add dynamics, articulations, fingerings and ornaments into your score.

Select the Zoom tool to magnify your score to facilitate the precise positioning of annotations.

The Clear button deletes all annnotations for the current page.

Tap on any toolbar button to get more information on the functions that it performs.
